## 🚀 快速开始

### 访问网站

- **本地开发环境**：http://localhost:8000
- **数据看板**：http://localhost:8000/
- **关于页面**：http://localhost:8000/about
- **API 文档**：http://localhost:8000/docs
- **健康检查**：http://localhost:8000/health

### 启动服务

```bash
# 进入项目目录
cd stock-dashboard

# 安装依赖（首次运行）
pip install -r requirements.txt

# 启动开发服务器
uvicorn app.main:app --reload

# 启动生产服务器
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## 📊 功能介绍

### 核心功能

#### 1. 基本面选股系统

系统基于四大维度对股票进行评分和排名：

| 维度 | 说明 | 关键指标 |
|------|------|----------|
| **成长能力** | 评估企业营收、利润的持续增长潜力 | 营收增长率、净利润增长率等 |
| **盈利能力** | 考察ROE、毛利率等盈利质量指标 | ROE、毛利率、净利率等 |
| **现金能力** | 分析经营性现金流、自由现金流的健康状况 | 经营性现金流、自由现金流等 |
| **偿债能力** | 评估资产负债结构及偿债风险 | 资产负债率、流动比率等 |

#### 2. Top 3 推荐股票

- 展示综合评分最高的 3 只股票
- 实时显示股票代码、名称、综合得分
- 点击可跳转到同花顺 F10 详情页

#### 3. 数据筛选

支持多维度筛选条件：
- 股票代码
- 股票名称
- 最小得分
- 最大得分
- 投资建议（强烈推荐、买入、持有、回避）

#### 4. 数据排序

支持按以下字段排序：
- 股票代码
- 股票名称
- 综合得分
- 成长能力得分
- 盈利能力得分
- 现金流得分
- 偿债能力得分
- 计算时间

#### 5. 数据搜索

- 快速搜索框支持股票代码和名称搜索
- 输入关键词实时显示搜索建议
- 支持键盘快捷键（Enter 确认，Escape 取消）

## 🎨 主题切换

### 主题介绍

系统提供三种主题：

| 主题 | 说明 | 图标 | 适用场景 |
|------|------|------|----------|
| **Teal 青色** | 默认主题，清爽明亮 | 🎨 | 日间使用，长时间工作 |
| **Red 红色** | 温暖色调，视觉冲击 | 🎨 | 强调重点，视觉活跃 |
| **Dark 暗黑** | 深色背景，护眼省电 | 🌙 | 夜间使用，减少眼部疲劳 |

### 切换方法

#### 桌面端
1. 点击顶部导航栏右侧的 **主题切换按钮**（调色盘图标）
2. 每次点击循环切换：Teal → Red → Dark → Teal
3. 切换时图标会旋转 360°

#### 移动端
1. 点击右下角的 **浮动主题切换按钮**
2. 每次点击循环切换
3. 主题设置会自动保存

### 主题持久化

- 主题选择保存在浏览器 `localStorage` 中
- 刷新页面后主题保持不变
- 清除浏览器数据会恢复默认主题（Teal）

## 📱 基本使用

### 查看股票列表

1. 进入首页（数据看板）
2. 浏览 Top 3 推荐股票卡片
3. 查看下方全部股票列表

### 筛选股票

1. 在筛选区域输入条件：
   - 股票代码：如 `000001`
   - 股票名称：如 `平安银行`
   - 最小得分：如 `80`
   - 最大得分：如 `100`
   - 投资建议：选择 `强烈推荐` 等
2. 点击 **应用筛选** 按钮查看结果
3. 点击 **重置** 按钮清除所有筛选条件

### 排序数据

1. 点击表头字段名（如"综合得分"）
2. 首次点击降序排序（↓）
3. 再次点击升序排序（↑）
4. 当前排序字段图标会高亮

### 搜索股票

1. 点击顶部搜索框
2. 输入股票代码或名称关键词
3. 选择搜索建议中的股票
4. 系统自动跳转并筛选显示该股票

### 查看股票详情

1. 点击任意股票行
2. 自动跳转到同花顺 F10 详情页
3. 在新标签页打开，不影响当前页面

### 翻页查看

1. 使用底部分页导航
2. 点击页码直接跳转
3. 点击 `«` 或 `»` 前后翻页
4. 查看总记录数显示在标题旁

## ❓ 常见问题

### 主题相关问题

#### Q: 如何切换主题？
A: 点击顶部导航栏右侧的调色盘图标，每次点击循环切换主题。

#### Q: 主题切换后刷新页面会重置吗？
A: 不会。主题选择保存在浏览器本地，刷新后保持不变。

#### Q: 为什么有些文字在暗黑主题下看不清楚？
A: 请清除浏览器缓存后刷新页面（Ctrl+Shift+R）。

### 数据相关问题

#### Q: 数据多久更新一次？
A: 股票基本面数据每日收盘后自动更新。

#### Q: 数据来源可靠吗？
A: 数据来自聚宽（JoinQuant）和 Tushare，是业内权威数据源。

#### Q: 评分系统是如何计算的？
A: 评分基于四大维度的多个财务指标加权计算，具体算法公开可查。

### 导航相关问题

#### Q: 关于页面无法打开怎么办？
A: 请检查浏览器控制台是否有错误，尝试清除缓存后刷新。

#### Q: 如何返回数据看板？
A: 点击侧边栏的"数据看板"链接，或点击页面的"首页"链接。

### 使用相关问题

#### Q: 搜索框如何使用？
A: 点击搜索框，输入股票代码或名称关键词，从下拉建议中选择，按 Enter 键确认。

#### Q: 筛选条件如何组合使用？
A: 可以同时使用多个筛选条件，系统会返回符合所有条件的结果。

#### Q: 如何重置所有筛选条件？
A: 点击"重置"按钮，清除所有输入和选择。
