- **无阴影**: `none`

## 🎯 核心设计特点

### 1. **毛玻璃效果升级**
- 侧边栏、顶栏: `rgba(255,255,255,0.4)` + `blur(24px)`
- 卡片、筛选: `rgba(255,255,255,0.6)` + `blur(12px)`
- 输入框: `rgba(255,255,255,0.6)` + `blur(12px)`
- 表格行: `blur(4px)`

### 2. **渐变色系统**
- Top 3 卡片每个排名使用不同的渐变色
- 得分使用从红到绿的渐变色
- 表头使用主题色到互补色的渐变
- 所有渐变都使用 135° 或 90° 角度

### 3. **视觉层次**
- 通过不同的 blur 值创建深度感
- 通过透明度创建层次感
- 通过阴影创建悬浮感
- 通过渐变色创建现代感

### 4. **交互反馈**
- 所有可交互元素都有 0.15s 过渡动画
- 使用 `cubic-bezier(0.4, 0, 0.2, 1)` 实现自然缓动
- 悬停时增加阴影和轻微缩放
- Focus 时增加边框颜色和阴影
